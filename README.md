> Frontend files and build tools for new miis.edu and others to come

## Requirements
- Node 6.4+

## Getting started


Install node modules:

```bash
npm install
```

### Development
Run local browser sync server and watch files:

```bash
npm start
```

### Build for production

```bash
npm run build
```
